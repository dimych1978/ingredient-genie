# **App Name**: Telemetron Sales App

## Core Features:

- Data Extraction: Extract sales data of an ingredient from telemetron.ru.
- Machine Selection: Let the user specify the number and types of machines to be considered
- Order List: Use a generative AI tool to determine the list of ingredients with amounts needed to produce for the given machines.
- Order List Display: Display the order list of ingredients with respective quantities required for production, formatted for easy readability.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), conveying trust, authority, and intelligence.
- Background color: Light Blue (#E3F2FD), a soft and clean background to ensure readability.
- Accent color: Amber (#FFC107), providing a point of emphasis to highlight important data and calls to action.
- Body font: 'Inter', a sans-serif font, for the main text. Headline font: 'Space Grotesk', a sans-serif font, for the headers.
- Use simple, clear icons from a set like Material Design Icons to represent different ingredients and actions.
- Maintain a clean, grid-based layout to present data and machine selections clearly.
- Subtle transitions and animations to provide feedback and improve the user experience when extracting data and generating order lists.
